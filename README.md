# Udhar Khata (Udhar Management Web App)

A lightweight, mobile-first digital khata (ledger) for tracking informal credit and debt transactions. 

![Project Banner](https://via.placeholder.com/800x400?text=Udhar+Khata+-+Digital+Credit+Management)

- **Dashboard**: Real-time view of total receivable and total payable amounts.
- **Customer Management**: Easily add, search, and manage customer records.
- **Transaction History**: Detailed ledger for each customer with "Gave Money" (Udhar) and "Got Money" (Vasuli) categories.
- **Data Backup**: Export your data as JSON and import it back anytime.
- **Privacy First**: All data is stored locally in your browser (LocalStorage). No backend required!
- **Modern UI**: Built with React and Vanilla CSS for a fast, responsive, and clean user experience.

## ğŸ“œ How to Use

1. **Add a Customer**: Click on "ADD CUSTOMER" and enter their name and phone number.
2. **Add a Transaction**: Select a customer from the list. Choose "GAVE MONEY" to record debt or "GOT MONEY" to record repayment.
3. **Backup Data**: Use the Download icon in the header to save a backup of your data. Use the Upload icon to restore it.
